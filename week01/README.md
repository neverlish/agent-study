# Week 01. Web Surfer

## 실행방법

- 의존성 설치
  `npm install`

- 필요 환경변수 설정

  - openapi key
    `- export OPENAI_API_KEY=${OPENAI API key}`
  - google search api
    - 참고: https://developers.google.com/custom-search/v1/using_rest?hl=ko
    - `export GOOGLE_SEARCH_ENGINE_ID=${ENGINE ID}`
    - `export GOOGLE_API_KEY=${API_KEY}`

- 실행

  - `npm run dev`
